#  Python Budget-Tracker

Dies ist mein allererstes Python-Projekt Es handelt sich um ein kleines Skript im Terminal, das dabei hilft, Einnahmen und Ausgaben zu tracken und am Ende visuell darzustellen.

##  Was das Programm macht:
1. Es fragt das monatliche Gehalt ab.
2. Es sammelt beliebig viele Ausgaben (Miete, Essen, etc.) über eine Endlosschleife, bis der Nutzer "x" drückt.
3. Es berechnet die Gesamtausgaben und das restliche Budget.
4. Es erstellt automatisch ein übersichtliches Kuchendiagramm (Pie Chart) mit den genauen Prozentanteilen.

##  Was ich in diesem Projekt gelernt habe:
- Den Umgang mit **Dictionaries**, um Datenpaare (Name der Ausgabe + Betrag) sauber zu speichern.
- Das Erstellen von **while-Schleifen**, um Programme interaktiv zu machen.
- **Hardcoding** und Kommentare zum effizienten Testen von Code.
- Die Integration und Nutzung von externen Bibliotheken wie `matplotlib` für die Datenvisualisierung.
